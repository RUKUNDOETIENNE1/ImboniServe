## 👥 **Customer CRM**

### **Understanding RFM Segmentation**

Your customers are automatically grouped into 6 segments:

#### **🏆 Champions** (Best Customers)
- **Who:** Recent, frequent, high-spending
- **Action:** Reward with VIP perks
- **Example:** Visited 3 times this month, spent 150,000 RWF

#### **💎 Loyal Customers**
- **Who:** Regular visitors, good spending
- **Action:** Keep them engaged
- **Example:** Visits monthly, consistent spending

#### **🌱 New Customers**
- **Who:** Recent first-time visitors
- **Action:** Welcome them, encourage return
- **Example:** First visit last week

#### **⚠️ At Risk**
- **Who:** Haven't visited recently
- **Action:** Win them back with offers
- **Example:** Last visit 3 months ago

#### **💔 Lost Customers**
- **Who:** Haven't visited in 6+ months
- **Action:** Re-engagement campaign
- **Example:** Last visit 8 months ago

#### **⭐ Promising**
- **Who:** Potential to become loyal
- **Action:** Nurture with targeted offers
- **Example:** 2-3 visits, increasing spending

### **How to Use CRM**

1. **View All Customers:**
   - Go to Dashboard → CRM
   - See all customers with their segments

2. **Filter by Segment:**
   - Click on a segment badge (e.g., "Champions")
   - View only that segment

3. **Search Customers:**
   - Use the search bar
   - Search by name, phone, or email

4. **Export Data:**
   - Click "Export" button
   - Download CSV for analysis

5. **View Customer Details:**
   - Click on any customer
   - See full history, orders, and spending

## 🏅 **Staff Performance**

### **Performance Metrics**

Each staff member is scored on:

1. **Orders Served** (30 points)
   - More orders = higher score
   - Target: 50+ orders/day

2. **Service Speed** (30 points)
   - Faster service = higher score
   - Target: < 2 minutes average

3. **Customer Rating** (25 points)
   - Based on customer feedback
   - Target: 4.5+ stars

4. **Tips Earned** (15 points)
   - Higher tips = better service
   - Target: 50,000+ RWF/day

**Total Score:** 0-100

### **Badge System**

- 🥇 **Gold:** 90+ score (Top performers)
- 🥈 **Silver:** 75-89 score (Great performers)
- 🥉 **Bronze:** 60-74 score (Good performers)
- ⭐ **Rising Star:** 50-59 score, 20+ orders (Improving)
- 🆕 **Rookie:** < 10 orders (New staff)

### **Leaderboard**

- **View:** Dashboard → Staff Performance
- **Period:** Today / This Week / This Month
- **Top Performer:** Highlighted at the top
- **Your Rank:** See where you stand

### **How to Improve**

1. **Serve More Orders:** Take more tables
2. **Be Faster:** Reduce wait times
3. **Be Friendly:** Get better ratings
4. **Provide Value:** Earn more tips

## 📅 **Reservations**

### **Creating a Reservation**

1. **Go to:** Dashboard → Reservations
2. **Click:** "New Reservation"
3. **Fill in:**
   - Customer name
   - Phone number (+250XXXXXXXXX)
   - Email (optional)
   - Party size (1-20 people)
   - Date and time
   - Table preference
   - Special requests

4. **Deposit:**
   - System calculates 20% deposit
   - Customer pays to confirm
   - Reduces no-shows

5. **Confirmation:**
   - Customer receives SMS with code
   - Code format: ABC123 (6 characters)

### **Reservation Status**

- **🟡 PENDING:** Waiting for deposit
- **🟢 CONFIRMED:** Deposit paid, reservation active
- **✅ COMPLETED:** Customer arrived and dined
- **❌ CANCELLED:** Customer cancelled
- **🚫 NO_SHOW:** Customer didn't arrive

### **Deposit Management**

- **PENDING:** Waiting for payment
- **PAID:** Deposit received
- **REFUNDED:** Cancelled, money returned
- **FORFEITED:** No-show, deposit kept

### **Managing Reservations**

1. **View All:** See upcoming reservations
2. **Filter:** By date, status, or table
3. **Update:** Change time, table, or details
4. **Cancel:** Refund deposit if needed
5. **Mark Complete:** When customer arrives

## 📦 **Inventory Alerts**

### **Alert Types**

#### **🔴 Critical (Red)**
- Stock will run out in < 2 days
- **Action:** Order immediately

#### **🟡 Warning (Yellow)**
- Stock will run out in 2-7 days
- **Action:** Plan to reorder soon

### **Setting Up Alerts**

1. **Go to:** Dashboard → Inventory Alerts
2. **Configure:**
   - Critical threshold: e.g., 10 units
   - Warning threshold: e.g., 20 units
   - Alert channels: Email, WhatsApp, Push

3. **Notification Channels:**
   - ✅ **Email:** Get alerts in inbox
   - ✅ **WhatsApp:** Instant messages
   - ✅ **Push:** Browser notifications

### **Responding to Alerts**

1. **Review Alert:** Check item and stock level
2. **Calculate Need:** Days until out
3. **Place Order:** Reorder from supplier
4. **Dismiss Alert:** Once handled

## 🧪 **A/B Testing**

### **What is A/B Testing?**

Test different versions of menu items to see which performs better.

**Example:**
- **Variant A:** Burger - 8,000 RWF
- **Variant B:** Burger - 7,500 RWF
- **Result:** See which price gets more orders

### **Creating a Test**

1. **Go to:** Dashboard → A/B Testing
2. **Click:** "Create Test"
3. **Select Item:** Choose menu item
4. **Create Variants:**
   - **Control:** Original version
   - **Variant:** New version
5. **Test:**
   - Price
   - Description
   - Image
6. **Set Traffic:** 50/50 split
7. **Duration:** 7-14 days recommended

### **Analyzing Results**

After test runs:
- **Views:** How many saw each variant
- **Orders:** How many ordered
- **Conversion:** % who ordered
- **Revenue:** Total sales
- **Winner:** Which performed better

### **Applying Winner**

1. **Review Results:** Check winner
2. **Click:** "Apply Winner"
3. **Confirm:** Update menu
4. **Done:** Winner becomes default

## 📱 **WhatsApp Campaigns**

1. **Go to:** Dashboard → Campaigns
2. **Click:** "New Campaign"
3. **Choose Segment:**
   - Champions
   - Loyal
   - At Risk
   - All customers

4. **Write Message:**
   - Use templates
   - Personalize with {name}
   - Add offer details

5. **Schedule:**
   - Send now

Enjoy 20% OFF your next visit. Show this message.

## 🔧 **Troubleshooting**

### **Can't Login?**

1. **Check email:** Make sure it's correct
2. **Reset password:** Click "Forgot Password"
3. **Clear cache:** Ctrl+Shift+Delete
4. **Try another browser:** Chrome, Firefox, Safari

### **Dashboard Not Loading?**

1. **Check internet:** Make sure you're online
2. **Refresh page:** Press F5
3. **Clear cache:** Ctrl+Shift+Delete
4. **Contact support:** support@imboniserve.com

### **Alerts Not Working?**

1. **Check settings:** Enable notifications
2. **Allow browser:** Grant permission
3. **Check email:** Not in spam folder
4. **WhatsApp:** Number is correct

### **Data Not Syncing?**

1. **Check connection:** Internet stable?
2. **Refresh page:** F5
3. **Logout/Login:** Fresh session
4. **Contact support:** If persists
